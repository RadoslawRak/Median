int Median ( int *  ArrayBuffer,

Calculates a median from passed samples. Sorts passed samples, so the operation is destructive to the data!

Works with odd and even lengths of data. For even - averages two central data points

Parameters
ArrayBufferData samples to calculate median from. Buffer indicates that this should be a copy of the original data, cause passed array will be altered (sorted).
LengthData array length
Returns
Median value of the passed data
